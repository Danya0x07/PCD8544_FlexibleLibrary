Detailed Description

Customizable font utility of the PCD8544 LCD controller library.

If the national locale support is required for your application, this file should be saved in an encoding in which national characters are represented by one byte.

◆ _get_character_bitmap()

static const uint8_t* _get_character_bitmap ( char  c)
static

Get the bitmap of a character.

The idea is to return the addresses of bitmap images of characters from different font tables depending on the selected language.

Note
This library works only with 5x7 fonts. Bitmaps can be programmatically scaled by the library.
Parameters
[in]cThe character whose bitmap to get.
Returns
Address of the bitmap of the character.

◆ ascii_charset

const uint8_t ascii_charset[][5]
static

ASCII character bitmap table.

if the project uses several libraries containing such a table, it can be moved from this file to a separate location, keeping it accessible from this file (#define ascii_charset your_custom_super_name_for_ascii_charset) so that there is no duplications. Other font tables also can be added here (or not here), but for this library only 5x7 fonts are supported (they can be scaled programmatically by the library).

Note
Additional architecture-specific attributes can be added to this declaration to tell the compiler to place this font table in the program memory.
