Detailed Description

The platform-specific part of the PCD8544 LCD controller library.

By implementing the interface declared in this file, you can port the library to any platform.

Note
In this file you can include any headers you need to implement the declared interface.

◆ _lookup()

static uint8_t _lookup ( const uint8_t *  pbyte)
inlinestatic

Look up a byte in the program memory.

This function is used to retrieve bytes of bitmaps that are placed in program memory. They are bitmaps from font tables and user created images, which have the pcd8544_image.lookup field set to true. If no additional operations are required to retrieve data from program memory, the return value of this function must be *pbyte.

For example, in case of AVR we should return pgm_read_byte(pbyte).

Parameters
[in]pbyteAddress of a byte being looked up.
Returns
The value of the byte with the given address.
