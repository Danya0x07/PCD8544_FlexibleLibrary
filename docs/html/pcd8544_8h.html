void pcd8544_draw_img ( uint8_t  x0,
uint8_t  p0,
const struct pcd8544_image *  img 
)

Draw an image on the display.

Printing scaled images is useful. For example, to print a simple logo that has no high-resolution details, we can save it to pcd8544_image with width_px == 14 and height_pg == 1, and then scale it 6 times by calling pcd8544_setup_brush(). Thus, it will fit the entire display, but it's bitmap will take 14 instead of 504 bytes of memory.

Parameters
[in]x0X coordinate of the left corner of the image [0 - 83].
[in]p0A page from which to start drawing the image [0 - 5].
Note
if the image does not fit on the display, the part that does not fit is not drawn.

Print a string on the display in a virtual frame.

With this function you can print a string in a limited rectangular area of the display.

Parameters
[in]x0Left border of the virtual frame [0 - 83].
[in]x1Right border of the virtual frame [0 - 83].
[in]p1Bottom border (page) of the virtual frame [0 - 5].
[in]sString to print.
Note
The p0 (top border) parameter is taken from last call to pcd8544_set_cursor() and it musst be less then p1, otherwise result is incorrect. And x0 < x1.

void pcd8544_set_power ( bool  pwr)

Control the PCD8544 power mode.

Entering power-down mode can reduce current consumption of the PCD8544.

Parameters
[in]pwr1 to select chip-active mode, 0 to select power-down mode.
Note
Before entering power-down mode, the PCD8544 DDRAM is filled with '0's to ensure specified current consumption. However, the framebuffer, if used, is not flushed.
